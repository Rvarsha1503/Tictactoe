# Tic-Tac-Toe Game

This is a simple command-line implementation of the classic Tic-Tac-Toe game in C. You can play against the computer in this game.

## How to Play

1. Run the program.
2. Enter any key to start the game.
3. The board will be displayed with empty spaces represented by blanks.
4. You are X, and the computer is O.
5. Enter the row and column number where you want to place your move (1-3).
6. The computer will make its move automatically.
7. The game continues until someone wins or it's a draw.
8. After the game ends, you will be asked if you want to play again.
9. Enter 'Y' to play again or 'N' to exit.

## Features

- Simple command-line interface.
- Ability to play against the computer.
- Automatic detection of winner or draw.
